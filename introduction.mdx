# Welcome to ts-caldav

**ts-caldav** is a modern TypeScript CalDAV client that makes it simple to interact with CalDAV servers. Whether you're building a calendar app, integrating with existing calendar systems, or synchronizing events across platforms, ts-caldav provides a clean, type-safe API that works seamlessly across Node.js, browsers, and React Native.

## Key Features

ts-caldav is designed to make CalDAV integration effortless while providing powerful features for advanced use cases:

### Authentication & Discovery

- **Credential Validation** - Automatically validates credentials against CalDAV servers
- **Endpoint Discovery** - Automatically discovers CalDAV endpoints using `.well-known/caldav` and common paths
- **Multiple Auth Methods** - Supports both Basic Auth and OAuth2 authentication
- **Compatible with Major Providers** - Works with Google Calendar, iCloud, Yahoo, Fastmail, Nextcloud, and more

### Calendar & Event Management

- **Calendar Operations** - Fetch calendar homes and individual calendars with full metadata including colors
- **Event CRUD** - List, create, update, and delete calendar events
- **Recurring Events** - Full support for creating and managing recurring events with complex rules
- **Todo Support** - Complete VTODO component support for task management
- **Timezone Aware** - Handle events across multiple timezones with `startTzid` and `endTzid`

### Efficient Synchronization

- **Change Detection** - Use `getctag` and `etag` to detect calendar and event changes efficiently
- **Diff-based Sync** - Only fetch what changed with intelligent sync algorithms
- **Cache Support** - Export and restore client state to avoid repeated discovery requests
- **Minimal Network Overhead** - Optimized queries reduce bandwidth usage

### Developer Experience

- **Full TypeScript Support** - Complete type definitions for all APIs
- **Promise-based API** - Modern async/await patterns throughout
- **Cross-platform** - Works in Node.js, browsers, and React Native
- **Comprehensive Documentation** - Rich examples and detailed API reference

## Simple Example

Here's how easy it is to get started with ts-caldav:

```typescript
import { CalDAVClient } from "ts-caldav";

// Create and authenticate a client
const client = await CalDAVClient.create({
  baseUrl: "https://caldav.icloud.com",
  auth: {
    type: "basic",
    username: "your-username",
    password: "your-app-specific-password",
  },
});

// Fetch calendars
const calendars = await client.getCalendars();
console.log(`Found ${calendars.length} calendars`);

// Get events from the first calendar
const events = await client.getEvents(calendars[0].url, {
  start: new Date(),
  end: new Date(Date.now() + 7 * 24 * 60 * 60 * 1000), // Next 7 days
});

console.log(`Upcoming events:`);
events.forEach(event => {
  console.log(`  - ${event.summary} at ${event.start}`);
});
```

## Supported CalDAV Servers

ts-caldav has been tested and works with the following CalDAV providers:

| Provider      | Endpoint Example | Notes |
|:--------------|:------------------|:------|
| **Google Calendar** | `https://apidata.googleusercontent.com/` | Requires OAuth2 or app-specific password |
| **iCloud** | `https://caldav.icloud.com/` | Requires app-specific password |
| **Yahoo** | `https://caldav.calendar.yahoo.com` | |
| **Fastmail** | `https://caldav.fastmail.com` | Requires app-specific password |
| **GMX** | `https://caldav.gmx.net` | |
| **Nextcloud** | `https://your-server.com/remote.php/dav` | Self-hosted |
| **Radicale** | `https://your-server.com/` | Self-hosted |

<Note>
Some providers require you to enable CalDAV access or generate app-specific passwords. Check your provider's documentation for setup instructions.
</Note>
